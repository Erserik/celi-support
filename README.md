## Частые вопросы / FAQ

**Где хранятся мои данные? / Where is my data stored?**
Все данные хранятся только на вашем устройстве. Приложение не требует регистрации и не передаёт данные на серверы.
All data is stored locally on your device. No registration, no cloud sync, no data collection.

**Как отменить подписку Celi Pro? / How do I cancel Celi Pro?**
Настройки → Apple ID → Подписки → Celi Pro → Отменить подписку.
Settings → Apple ID → Subscriptions → Celi Pro → Cancel Subscription.

**Как удалить все данные? / How do I delete all my data?**
Удалите приложение — вместе с ним удаляются все данные.
Delete the app — all data is deleted with it.
